# ПРОГРАММУЛЕЧКА
Для того, чтобы интерфейс работал, нужна библиотека PyQt5

Установить можно так:
`pip install pyqt5'`

Запускаем, работаем.

## Управление

Всё просто.
```
w -     Движение вперёд
s -     Движение назад
a -     Поворот сервопривода влево
d -     Поворот сервопривода вправо
shift - Повысить скорость для движения вперёд
ctrl -  Понизить скорость для движения вперёд
```
Менять скорость при движении назад пока нельзя
