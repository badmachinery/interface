# ПРОГРАММУЛЕЧКА
Для того, чтобы интерфейс работал, нужна библиотека PyQt5 (на ней работает интерфейс)

Установить можно так:
`pip install pyqt5`

Еще нужна библиотека paramiko (на ней работает SSH соединение)

Устанавливаем
`pip install paramiko`

## Кнопки
1. OFF/ON - Подключиться по SSH (а вот отключиться пока нельзя)
2. Запустить скрипт - Запускает скрипт на распберри, начало работы
3. Загрузить файл - Выбираем файл на ноутбуке, и он отправляется в рабочую папку на распберри, используем чтобы закидывать на нее новый код
4. Кнопка специально для Германа - Запускает на ноутбуке хотспот (если он еще не запущен)

Есть SSH консоль, типа как в путти, туда можно писать команды в моменты когда скрипт не запущен, а во время работы скрипта туда копируются все принты, которые делает распберри

## Управление

Всё просто.
```
w -     Движение вперёд
s -     Движение назад
a -     Поворот сервопривода влево
d -     Поворот сервопривода вправо
shift - Повысить скорость для движения вперёд
ctrl -  Понизить скорость для движения вперёд
r -     Затормозить, работает пока так себе, но для крайнего случая сойдет
k -     Перезагрузить Arduino
p -     Запустить автоскрипт (так себе название, но лучше не придумал) на выполнение
c -     Безопасно выключить программу на распберри, после выключения программу можно перезапустить
```
