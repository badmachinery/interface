# ПРОГРАММУЛЕЧКА
Для того, чтобы интерфейс работал, нужна библиотека PyQt5 (на ней работает интерфейс)

Установить можно так:
`pip install pyqt5'`

Еще нужна библиотека paramiko (на ней работает SSH соединение)

Устанавливаем
`pip install paramiko`

Запускаем, работаем.

## Управление

Всё просто.
```
w -     Движение вперёд
s -     Движение назад
a -     Поворот сервопривода влево
d -     Поворот сервопривода вправо
shift - Повысить скорость для движения вперёд
ctrl -  Понизить скорость для движения вперёд
c -     Выключить программу на распберри (интерфейс остается активен)
```

## Проблемы
1. Менять скорость при движении назад пока нельзя
2. Выключение программы на распберри не имеет особого смысла, из интерфейса её перезагрузить всё равно нельзя
